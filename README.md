# Credit Card UI Design in HTML & CSS | Card Design

Um projeto simples de **design de cartão de crédito** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **cartão de crédito moderno** com layout elegante e estilização usando apenas CSS.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Design moderno de cartão
- Layout organizado
- Fácil de personalizar

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
